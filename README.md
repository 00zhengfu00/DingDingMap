##**钉钉地图项目是一个基于android 5.0以上，MatetialDesign设计的项目。**

###**实现的功能**
-**5.0新特性揭露动画、共享元素**
-**定位功能**
-**实时路况信息**
-**地点查询**
-**公交路线推荐**
-**驾车路线推荐**
-**步行路线推荐**
-**骑行路线推荐**
-**离线地图功能**
-**路线规划功能**
-**四种地图模式**
-**分享功能**
-**美女福利**
